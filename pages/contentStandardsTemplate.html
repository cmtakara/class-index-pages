Themes

Themes are related to the topics that are covered, for example solving linear equations or the Civil War. There are certain themes that make sense to cover prior to others, but the order can be changed to fit student needs. Practice can be linked based on the lessons that are planned.


# Text Lesson Linked Practice
1 Overarching Theme ...
1.a sub-theme Module x lesson x (link) Practice x.y
1.b next sub theme Module x lesson y (link) Practice w.v

Practices

Practice are related to specific types of skills and can be across lessons. They can even be applied across subjects. For example, many reading practices are applied to Social Studies. Or, a practice from science is measure of central tendency, which is a math lesson.


# Text Lesson Linked Theme
1 Overarching Practice ...
1.a subset of skills Module x lesson x (link) Theme x.y
1.b next subset of skills Module x lesson y (link) Theme w.v
